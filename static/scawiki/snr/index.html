The SNR (Signal-to-Noise Ratio) is a measure of the signal strength relative to the noise level. To calculate it, we need to define two things:

  1. What constitutes the Signal?
  2. What constitutes the Noise?

While these definitions may seem straightforward, calculating SNR in a way that depends on the data requires more than just considering all the measured traces together. We must first split the traces into different classes or groups.

How we split the traces depends on the leakage model being used. Once we have divided the traces into groups, we can calculate the signal as the variance of the means within each class, and the noise as the mean of the variance within each class.

Calculating SNR without a Leakage Model

Important

Step 2 is done on one byte of the sbox output. i.e. SBOX(pt ^ key) will produce a 16 byte value (assuming AES-128), we only select one byte on which we calculate the SNR for.

The process can be summarized in three steps:

  1. Initialize arrays (256 x n_samples) to store per-byte statistics
  2. For each possible S-box output byte (0-255):
    1. Select traces with matching byte value
    2. Compute mean and variance across selected traces
  3. Calculate SNR as: (Variance of per-byte means) / (Mean of per-byte variances)

This can be done in code as follows.

# Initialize storage arrays
group_means = np.zeros((256, n_samples))
group_vars = np.zeros((256, n_samples))

# Compute per-byte statistics
for byte_val in range(256):
    idx_mask = (labels == byte_val)
    group_means[byte_val] = np.mean(leakage_traces[idx_mask], axis=0)
    group_vars[byte_val] = np.var(leakage_traces[idx_mask], axis=0)

# Calculate final SNR components
signal = np.var(group_means, axis=0)  # Between-class variance
noise = np.mean(group_vars, axis=0)  # Within-class variance
snr = signal / noise

# Visualize SNR profile
px.line(snr).show()

Calculating SNR with a Leakage Model

The process is similar, with the main difference being in how we group the traces. Instead of grouping them by the 256 possible byte values the intermediate value can take, we group them by the possible values that our power model outputs, which are based on the Hamming weight. For Hamming weight, these values range from 0 to 8, meaning we will have 9 groups or classes instead of the original 256.

group_means = np.zeros((9, n_samples))
group_vars = np.zeros((9, n_samples))

for i in range(9):
    idx_mask = np.nonzero(HW[labels] == i)
    group_means[i] = np.mean(leakage_traces[idx_mask], axis=0)
    group_vars[i] = np.var(leakage_traces[idx_mask], axis=0)

signal = np.var(group_means, axis=0)
noise = np.mean(group_vars, axis=0)


snr = signal / noise

px.line(snr).show()

Calculating SNR with SCALib

Here’s a sample code[1]:

snrobj = SNR(256,traces.shape[1],1)
snrobj.fit_u(
    traces.astype(np.int16),
    classes_idx.astype(np.uint16)
)
snr_val = snrobj.get_snr()[0]
return snr_val
