Correlation Power Analysis is simple in theory, it gets a little tricky when we start implementing it for a couple of reasons in my opinion:

  1. we need use pearson’s correlation coefficient which uses covariance matrices and using the high level numpy implementation np.corcoeff() is very slow, so we need to implement it ourselves with matrix operations.
  2. we need to write the proper flow of the attack which is about 2-3 nested loops.

We will tackle each of these challenges one at a time and hopefully it will all make sense in the end.

def recover_key_cpa(traces, plaintexts, SBOX=SBOX, HW=HW):
    """
    Performs Correlation Power Analysis (CPA) to recover AES key bytes.
    
    Parameters:
        aligned_traces (np.ndarray): Aligned power traces of shape (n_traces, n_samples)
        plaintexts (np.ndarray): Corresponding plaintexts of shape (n_traces, 16)
        SBOX (np.ndarray): AES S-box lookup table
        HW (np.ndarray): Hamming weight lookup table
    
    Returns:
        np.ndarray: The recovered key (16 bytes)
    """
    n_traces, n_samples = traces.shape
    
    # Precompute statistics for the traces
    A = traces
    A_mean = np.mean(A, axis=0)
    A_centered = A - A_mean
    var_A = np.sum(A_centered ** 2, axis=0)
    
    # Store recovered key bytes
    recovered_key = np.zeros(16, dtype=int)
    
    for bnum in trange(16, desc="Recovering Key Byte"):
        p = plaintexts[:, bnum]  # Get plaintext values for this byte
        key_guesses = np.arange(256)  # All possible key byte values
        
        # Compute intermediate values: p XOR key_guess -> SBOX -> Hamming weight
        xor_result = np.bitwise_xor(key_guesses[:, None], p[None, :])
        hyp_matrix = HW[SBOX[xor_result]]  # Shape: (256, n_traces)
        
        # Compute mean and variance for each hypothesis vector (each row)
        Y_mean = np.mean(hyp_matrix, axis=1)
        Y_centered = hyp_matrix - Y_mean[:, None]
        var_Y = np.sum(Y_centered ** 2, axis=1)
        
        # Compute Pearson correlation
        numerator = Y_centered @ A_centered
        denom = np.sqrt(var_Y[:, None] * var_A[None, :])
        corr_matrix = numerator / denom
        
        # Find best key guess based on max absolute correlation
        max_corrs = np.max(np.abs(corr_matrix), axis=1)
        best_key = np.argmax(max_corrs)
        recovered_key[bnum] = best_key
        
        print("".join(chr(k) for k in recovered_key[:bnum+1]), end="\r")
    
    print("\nRecovered Key:", recovered_key)
    return recovered_key
