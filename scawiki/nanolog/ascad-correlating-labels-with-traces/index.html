ASCAD: correlating labels with traces?

Something interesting which I’ve thought of doing is to find the correlation between the labels (which is really the sbox(pt[3] ^ key[3])) and the traces.

What this achieves is finding the sample point with the most correlation coeffecient which could be a better way to quantify leakage than using SNR?

I don’t really fully understand this but thought I would write this down anyway.

Here’s the code:

corrs = np.zeros((n_samples, 1))
for sample_idx in trange(n_samples):
    intermediates = labels[:, np.newaxis]
    corr = np.corrcoef(leakage_traces[:, sample_idx], intermediates.T, rowvar=True)
    corrs[sample_idx] = corr[0][1]
